🧠 DSA Project Using C++

A collection of Data Structures & Algorithms based mini-projects implemented in C++.
This project demonstrates problem-solving, logic building, and core DSA concepts through interactive console applications.

📂 Project Structure

│
├── MEMORY BHAI/
│   ├── main.cpp
│   ├── game.cpp
│   └── game.h
│
├── SUDOKU BHAI/
│   ├── main.cpp

🚀 Features
🔹 Maze Solver using queue (BFS approach)
🔹 Memory Game implementation
🔹 Sudoku Solver using Backtracking
🔹 Modular and structured C++ code
🔹 Multiple mini-projects in one repository
🛠️ Technologies Used
C++
Data Structures & Algorithms
Object-Oriented Programming (OOP)
Standard Template Library (STL)
▶️ How to Run
Clone the repository:
git clone https://github.com/Aman2400650/DSA-PROJECT-USING-C-.git
Navigate to the project folder:
cd DSA-PROJECT-USING-C-
Compile any project:

Example (Sudoku Solver):

g++ SUDOKU\ BHAI/main.cpp SUDOKU\ BHAI/sudoku.cpp -o sudoku
Run:
./sudoku
